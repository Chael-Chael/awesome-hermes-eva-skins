# Fonts

This directory includes the Ark Pixel 12px Monospaced TTF files used by the Windows Terminal profile in this repo.

Source:

- Project: https://github.com/TakWolf/ark-pixel-font
- Release: `2026.05.07`
- Asset: `ark-pixel-font-12px-monospaced-ttf-v2026.05.07.zip`
- Release asset SHA256: `3356ca96846b06664879158a1f25c6de7ad43d52046b8b0ee6e2d90e4512d1d8`

Bundled files:

- `ark-pixel-12px-monospaced-latin.ttf`
- `ark-pixel-12px-monospaced-zh_cn.ttf`
- `ark-pixel-12px-monospaced-ja.ttf`
- `ark-pixel-12px-monospaced-ko.ttf`
- `OFL.txt`

Only these four TTF files are bundled because they are the font families referenced by `windows-terminal/cool-retro-frame-amber.profile.jsonc`.
