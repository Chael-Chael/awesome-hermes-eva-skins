## Windows Terminal shader references

`shaders/cool-retro-frame-amber.hlsl` is a custom amber CRT shader based on the Windows Terminal `samples/PixelShaders/Retro.hlsl` sample and designed to be used with the Windows Terminal pixel shader setting.

The surrounding shader workflow is inspired by:

- Hammster/windows-terminal-shaders: https://github.com/Hammster/windows-terminal-shaders
- Microsoft Terminal sample shaders: https://github.com/microsoft/terminal/tree/main/samples/PixelShaders

The upstream `Hammster/windows-terminal-shaders` repository is distributed under CC0-1.0. Microsoft Terminal sample code is distributed under the license used by the Microsoft Terminal repository. Keep these notices if you redistribute or modify the shader.
